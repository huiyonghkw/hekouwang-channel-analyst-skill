# 04 · 付费服务化（参考，非承诺）

> 这是把工具变成生意的参考方案，数字是市场锚点，按你定位调。**先自用跑顺、攒够样板案例再对外卖。**

## 免费内核 vs 增值付费（边界 ⭐）

| | 免费（开源内核） | 增值（付费） |
|---|---|---|
| 是什么 | `pull/analyze/build_report` 三件套 + 自用版报告 HTML | 人的解读 + 品牌报告 + 配套内容生产 |
| 谁用 | 任何人 `git clone` 到 `~/.claude/skills/` 本地跑 | 找 @huiyonghkw 下单的客户 |
| 范围 | 拉自己账号数据、出一份自用复盘图 | 见下「增值三层」 |

**一句话口径**：**跑工具免费；"看得懂的解读 + 好看的品牌报告 + 照着改的内容生产"是付费。**

### 增值三层（每层叠加，靠组合其它 skill 交付）

1. **解读层**（人的价值）：把报告里的数据翻成"下一步具体做什么"，单次诊断 / 周报 / 月度陪跑。工具给数据，人给判断——这是机器替不了、最该收费的部分。
2. **品牌报告层**：客户版报告（落款换服务方品牌 + 联系方式 + CTA），用 `hekouwang-content-factory` 的私有品牌字体/版式（不随开源仓库分发）。实现：`build_report.py --mode client`（见 `03-报告设计.md`）。
3. **内容生产层**（接着诊断往下做）：诊断指出"封面要返工/选题要换"后，直接承接生产——
   - 封面返工 / 小红书图文 / 头条 → `hekouwang-content-factory`
   - 财经类账号数据报告 → `hekouwang-stock-data-reader-skill`（守金融合规）
   - 把复盘做成演示版 → `hekouwang-yandu-deck-skill`
   "诊断 → 生产" 一条龙，是最高客单价的组合拳。

> 边界铁律：**开源仓库只放免费内核**（不含私有字体、不含客户数据）；增值部分靠"人 + 私有资产 + skill 组合"交付，不进公开仓库。

## 产品阶梯（从轻到重）

| 套餐 | 交付物 | 适合 | 定价锚点（参考） |
|---|---|---|---|
| 单次账号诊断 | 1 份复盘报告 HTML + 一页核心建议 | 想体检账号的个人/小商家 | ¥99–299/次 |
| 数据周报订阅 | 每周 1 份报告（自动取数）+ 趋势对比 + 本周动作清单 | 持续运营的博主/品牌号 | ¥199–499/月 |
| 月度陪跑 | 周报 + 每月 1 次 1v1 复盘 + 选题/封面方向建议 | 认真做增长的账号 | ¥1k–3k/月 |
| 多账号/矩阵 | 批量报告 + 横向对比看板 | MCN/矩阵号/代运营同行 | 按账号数阶梯报价 |

> 定价逻辑：**单次低价引流 → 周报订阅做留存 → 陪跑做客单价**。先用单次诊断把人引进来。

## 为什么有人付费（卖点）

1. **省时间**：自己看后台一堆数字看不懂；一份报告直接告诉你"卡在哪、下一步做什么"。
2. **有方法**：漏斗诊断 + 出池逻辑 + 分优先级建议，不是"多发点""加油"这种废话。
3. **可视化体面**：V2 米白报告比截图后台专业得多，自己也能转发当成长记录。
4. **守底线反而是卖点**：不刷量、不承诺数字——长期可信，区别于满天飞的"包涨粉"骗局。

## 获客（冷启动）

- **自己账号当样板**：把"用这套从 301 起步"的真实复盘报告做成案例图，发小红书/朋友圈，本身就是销售物料。
- **免费体检引流**：前 N 个粉丝免费出一份诊断报告，换口碑 + 真实案例授权。
- **同行/同城博主**：数据焦虑最强的人群，从身边博主社群切。

## 交付与话术

- 交付：HTML（在线看）+ PDF（存档）+ 核心建议一页纸（微信直接发）。
- 边界话术（先说清，避免纠纷）：
  > "这份报告基于你账号的公开后台数据做中立复盘，给的是**方向和动作建议**，不承诺具体涨多少粉——增长取决于你执行+内容+平台，我帮你把'该做什么'看清楚。"
- 取数要客户**自己授权**（他登录他的后台/给只读权限），不碰账号密码、不替客户发任何内容。

## 合规与风险

- 数据只来自客户授权的公开创作者后台；不抓非授权数据、不碰他人隐私。
- 不做"包涨粉/保排名"承诺；不提供刷量。
- 客户版报告底部保留"数据来源 + 不构成未来表现承诺"说明条。
